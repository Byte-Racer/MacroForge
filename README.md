# MacroForge

MacroForge is a personal, minimally gamified nutrition tracker built with React + Vite.

It is intentionally a fun solo project for one person, focused on daily meal logging, macro tracking, water tracking, and light gamification.

## Tech
- React + Vite
- Tailwind CSS
- React Router
- Recharts
- localStorage persistence

## Note
This is not a multi-user or production-grade health platform. It is a personal side project built for enjoyment and consistency.
